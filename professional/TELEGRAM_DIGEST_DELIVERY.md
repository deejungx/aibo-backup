# Daily Digest Telegram Delivery

## Current Setup

The morning digest is generated by a cron job at 7 AM and delivered to Telegram via the `message` tool with the following configuration:

**To prevent link previews:**
```
message tool settings:
- action: send
- channel: telegram
- target: [deejung's telegram id]
- message: [digest content]
- disable_link_preview: true  (or link_preview_options parameter)
```
