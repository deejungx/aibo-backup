## What We Discussed

### Repos Found & Analyzed

1. **agent-studio** (Next.js frontend)
   - Multi-step wizard for creating AI agents
   - Split-stack: Supabase auth + external PostgreSQL
   - Organization + membership model
   - QR code generation via Symbiosis API
   - Superuser admin mode for full platform visibility

2. **agent-api** (Python FastAPI)
   - WhatsApp trigger endpoint
   - Task queuing to Redis
   - Recent work: polling-based instead of webhooks
   - API v1 (legacy) and v2 (crew agents)

3. **agent-runtime** (Celery workers)
   - Async task execution
   - RAG with embedding models
   - Crew-based agent orchestration
   - Structured response handling

### Data Flow (Complete)
- User scans QR → agent-api receives trigger
- API queries PostgreSQL for agent config
- Task enqueued to Redis
- Celery worker picks up → executes Crew agent
- Response stored + delivered to WhatsApp
